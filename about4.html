常見問題

  • Q1: 想快速了解「數位出版品資訊網」之服務內容及使用方式？

    Ans: 建議您先瀏覽首頁之「新手上路」或查看首頁之「網站導覽」功能。

  • Q2: 「數位出版品資訊網」提供那些服務？

    Ans: 使用者可於本系統查詢圖書之「書目」、「簡介」、「篇章」、「出版資訊」等，並申請「線上閱讀」、「線上列印」、「POD全書列印」等服務。

  • Q3 : 「數位出版品資訊網」有那些特色？
    Ans:
    • 一、即時傳遞：數千本精選學術圖書與書籍全文影像，線上即時傳遞!
    • 二、持續擴充：圖書內容具知識性、學術性、價值性，持續定期新增!
    • 三、服務多元：
      1. 提供會員線上「查詢」、「閱讀」、「列印」各圖書章節內容
      2. 精選圖書提供POD全書列印服務
      3. 貼心的個人化查詢、我的書架等功能
      4. 圖書精選推薦
      5. 新書主題介紹
      6. 各項排行統計
      7. 會員優惠活動
    • 四、使用便利：
      1. 書目快速檢索
      2. 線上e覽全書
      3. 單書多人閱讀
      4. 即時章節列印
      5. 閱讀次數不限

  • Q4 : 「數位出版品資訊網」收錄那些資料內容？

    Ans: 本系統完整收錄國家教育研究院歷年出版之圖書（含主編譯、學術絕版書、特色學科用書、參考書等各類出版品），提供民眾查詢各圖書之「書目」、「簡介」、「篇章」、「出版資訊」等資料。

  • Q5 : 「數位出版品資訊網」提供那些圖書種類？
    Ans:
    1. 知性【人文社會叢書】：知古今、賞文學、看故事、學語文……
      人文社會科學類學術著作、研究用書。包含：歷史、文學、故事、語文等。
    2. 探索【自然科學叢書】：精數理、通化學、玩科技、查術語……
      包含數學、物理、電子、化學等學術名詞、工具書。
    3. 典藏【中華叢書】：閱古籍、賞詩詞、談經文……
      中華古籍之編纂與整理。如：詩詞、經文、論叢等。
    4. 學術【大學用書】：一覽自然理工、文史法商等大學用書及專門研究……
      包含自然、理工、法商、文史等學術著作及專門研究。
    5. 精選【世著譯本】：原貌呈現各國經典、文學名著等……
      各類外文學術著作。如：科學、歷史、文學名著等。

  • Q6 : 「數位出版品資訊網」資料內容之收錄年限？

    Ans: 收錄國家教育研究院自民國44年起～迄今出版之各類學術圖書。

  • Q7 : 「數位出版品資訊網」資料內容之更新頻率？

    Ans: 每年持續新增圖書出版資訊及電子全文影像內容。

  • Q8 : 如何使用「數位出版品資訊網」？
    Ans:
    1. 免費使用：一般使用者可於本系統免費查詢圖書之「書目」、「簡介」、「篇章」、「出版資訊」等並「訂閱電子報」。
    2. 付費使用：如欲申請圖書章節之「線上閱讀」、「線上列印」或「POD全書列印」等服務，需先繳交年度會費，並依規定使用。請參見新手上路喔！

  • Q9 : 「數位出版品資訊網」資料內容之提供及系統製作之財產權？

    Ans: 本系統資料內容之著作財產權為「國家教育研究院」所有；授權委託「凌網科技」開發營運，任何形式之轉載及利用，請先與我們聯繫!

  • Q10 : 「數位出版品資訊網」有那些功能？
    Ans:
    1. 圖書查詢：提供「書名」、「作者」、「關鍵詞」等多種檢索點，並可使用「簡易」、「進階」、「瀏覽」等查詢方式。
    2. 新書上架：最新上架圖書介紹及瀏覽近期上架圖書。
    3. 推薦精選：推薦各主題專書做內容介紹。
    4. 暢銷排行：瀏覽本站電子書利用之各項排行統計。
    5. 會員專區：提供會員各項專屬及查詢服務。
    6. 最新消息：不定期報導最新優惠方案及活動內容。
    7. 新手上路：完整介紹本系統之各項服務內容，讓新手們更得心順手。
    8. 電子報：每月定期發送電子報，報導活動快訊、精選推薦等訊息。
    9. POD全書列印：提供使用者POD訂購服務，依據訂購需求產生印製。

  • Q11 : 已查到所需的圖書章節資料，該如何看到電子全文內容？

    Ans: 如欲「線上閱讀」所需之圖書章節電子全文內容，請您先加入會員、繳交年度會費並安裝HyView影像瀏覽軟體，再點選申請文獻傳遞之功能圖示。
